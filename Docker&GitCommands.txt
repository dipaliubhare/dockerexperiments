#to use docker
1.write insturction in Dockerfile-
-->to create Dockerfile=>by using notepad save as "Dockerfile" extension=> All
2. to create a image
-->docker build -t "any-name:tag" (note=> name should bw small case)
3.to create container
-->1.docker run "imagename:tag"
-->2.docker container run -d -p 8000:9000 "hello-program:latest"
4.for interactive terminal
-->docker run -it "imagename:tag"
5.to stop container
-->docker stop cantainer <container id> (first 2-3 digit)
6.to see log of container
//optional
-->docker log <container id> (first 2-3 digit)
7.you can start a stopped containner like this too
-->docker start <container id> 

**8.to remove all stopped containners
-->docker container prune

9.to remove container
-->docker container rm <container id> (first 2-3 digit)

10.to give name to a container
-->docker container run -d --name hellocontainer -p 8000:9000 "hello-program:latest"


11. to pun xomething from net
-->docker pull tomcat:latest <--example

#to open in bash
-->docker exec -it "container name" /bin/bash
